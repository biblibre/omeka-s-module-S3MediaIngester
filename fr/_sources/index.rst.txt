S3 Media Ingester documentation
===============================

`S3 Media Ingester`_ adds a media ingester for files stored on S3 compatible
object storages.

.. _S3 Media Ingester: https://github.com/biblibre/omeka-s-module-S3MediaIngester

Installation
------------

See general end user documentation for `Installing a module
<http://omeka.org/s/docs/user-manual/modules/#installing-modules>`__.


Compatibility with other modules
--------------------------------

S3 Media Ingester can be used as a media source for `CSVImport <https://github.com/omeka-s-modules/CSVImport>`_
